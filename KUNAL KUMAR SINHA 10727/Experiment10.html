Using 'div' for Layout

About This Card

This entire white box (including this text and the heading) is inside a single 'div' element. We use it to create a layout component that we can style as one unit.

Inside this paragraph, we can use a 'span' element to target specific words for styling. Notice how the 'span' does not create a new line; it stays *inline* with the text.
